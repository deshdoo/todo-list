# Todo List

Учебный проект с SQLAlchemy и SQLAlchemy ORM.

## Описание
Консольное приложение для управления заметками. Поддерживает добавление, выполнение и удаление задач с сохранением в БД.
Запрашивает 3 заметки и сохраняет в БД.

## Функции
- 'add_todo' - добавление заметки в БД;
- 'mark_done' - отметить заметку как выполненную;
- 'delete_todo' - удаление заметки из БД;

## Запуск
```
pip install sqlalchemy
python todo.py
```
